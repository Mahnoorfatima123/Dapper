# Dapper Demo 

A simple .NET application demonstrating **CRUD operations** and **caching** using [Dapper](https://github.com/DapperLib/Dapper), a high-performance micro ORM for .NET.

## **Features**  
✅ Fast and lightweight ORM with Dapper  
✅ CRUD operations (Create, Read, Update, Delete)  
✅ Caching for optimized performance  
✅ Uses SQL Server as the database  

---

## **Technologies Used**  
- **ASP.NET Core 6+**  
- **Dapper ORM**  
- **SQL Server**  
- **MemoryCache / Redis (for caching)**
